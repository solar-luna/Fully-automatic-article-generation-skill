# 评分标准

## 内容类型识别

### new_tool（新工具/新模型介绍）
**关键词**：发布、推出、开源、新工具、新模型
**来源**：GitHub、Hugging Face、OpenAI、Anthropic
**优先级**：高

### tutorial（实战教程/技巧）
**关键词**：教程、技巧、如何、实战、指南、入门
**来源**：Medium、Dev.to
**优先级**：中

### industry_news（行业动态分析）
**关键词**：报告、研究、分析、趋势
**来源**：The Verge、TechCrunch、VentureBeat
**优先级**：低

## 评分维度

1. **技术深度（30分）**
   - 包含代码/API/教程：30分
   - 包含技术细节/算法：20分
   - 纯新闻：10分

2. **新颖性（25分）**
   - 新发布/开源：25分
   - 重要更新：18分

3. **实用性（25分）**
   - 有应用案例：25分
   - 实用工具/框架：20分

4. **热度（20分）**
   - GitHub >1000 stars：20分
   - GitHub >100 stars：14分
   - 其他：10分
